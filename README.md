# Login Page UI

A modern and responsive Login Page UI developed using HTML and CSS as part of the Synent Technologies Web Development Internship Program.

## 📌 Project Overview

This project is a clean and user-friendly login interface designed with modern UI principles. It includes essential login components and a responsive layout for different screen sizes.

## ✨ Features

* Email input field
* Password input field
* Login button
* Forgot Password link
* Responsive design
* Modern and clean user interface

## 🛠️ Technologies Used

* HTML5
* CSS3
